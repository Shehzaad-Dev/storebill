# StoreBill 📊

Offline Invoice & Khata Management App

## Features

* Create invoices offline
* Export PDF invoices
* Manage khata/ledger records
* No login required
* Fully offline-first

## Download

👉 Latest APK:
Go to **Releases → Download APK**

## Install

1. Download APK
2. Enable "Install unknown apps" on your device
3. Open the APK and install

## Privacy

All data is stored locally on device. No server upload.

## Notes for contributors

This repository contains the Flutter project and release artifacts. Build with the Flutter SDK and follow standard Flutter build steps.
